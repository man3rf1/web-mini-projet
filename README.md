# Web Mini Project - To-Do List

## Description
Ce projet est une application web simple de type To-Do List
développée en HTML, CSS et JavaScript.
Elle permet d'ajouter et de supprimer des tâches.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- Git / GitHub

## Instructions d'exécution
1. Cloner le dépôt
2. Ouvrir le fichier `index.html` dans un navigateur

## Fonctionnalités
- Ajouter une tâche
- Supprimer une tâche
- Interface simple et responsive
